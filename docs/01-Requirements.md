# Requirements

## Overview
- the application should ingest raw candlestick data and process it to identify market structures
- using the candlestick data the application should build and internal model identifying key market structures and allowing querying of such structures through interfaces
- the application should implement trading strategies incorporating the identified market structures
- the application should implement a back testing engine to test and validate different strategies


## Requirements breakdown
- identify key market structures from candlestick data
    - uptrend and downtrend
    - break of structure (BOS)
    - fair value gap (FVG)
    - order blocks (OB)
    - breaker blocks (BB)
    - supply zone and demand zone
- identify market structures for multiple time frames simultaneously
